Queries

Natural language query execution

Natural Language Query

Submit a natural-language request and get back a curated time series response.

Authorizations:
ApiKeyAuth

Responses

https://api.qoery.com/v0/query/nl

Request samples

  • cURL
  • Python
  • JavaScript

curl -sS "https://api.qoery.com/v0/query/nl" \
  -H "X-API-Key: your-api-key" \
  -H "Content-Type: application/json" \
  -d '{"query": "population of France"}'

Web Scraping

Web scraping and data extraction

Structured Web Scrape

Download a web page, and structured data as time series. Use query params html=true and/or markdown=true to include raw artifacts inline; otherwise artifact URLs are provided in artifacts.

Authorizations:
ApiKeyAuth
query Parameters
html
boolean
Default: false

Include page HTML inline in response

markdown
boolean
Default: false

Include page markdown inline in response

Responses

https://api.qoery.com/v0/scrape

Request samples

  • cURL
  • Python
  • JavaScript

curl "https://api.qoery.com/v0/scrape" \
  -H "X-API-Key: your-api-key" \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com/statistics"}'
