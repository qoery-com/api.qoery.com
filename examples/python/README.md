# Python Examples

Python examples using the Qoery SDK.

## Setup

```bash
pip install qoery
```

## Examples

- `datasets_example.py` - Dataset collection jobs (start, poll, download CSV)
- `usage_example.py` - Usage statistics
- `plans_example.py` - Subscription plans and pricing

## Usage

1. Replace `YOUR_API_KEY` with your actual API key
2. Run: `python example_name.py`

## Notes

All examples use the production API endpoint `https://api.qoery.com`
